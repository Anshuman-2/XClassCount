# XCounterClass React App

This is a class-based Counter application built with React.

## How to Run

1. Install dependencies:
   npm install

2. Start the development server:
   npm start

3. Open http://localhost:3000 in your browser to view the app.

## Features
- Increment and decrement the counter
- Counter can go negative
- Buttons are named "Increment" and "Decrement"
- Count is displayed as "Count: valueOfCount"
